# Security Policy

## Supported Versions

We provide security updates for the following versions:

| Version | Supported          |
| ------- | ------------------ |
| v4.2.x  | :white_check_mark: |
| v4.1.x  | :x:                |
| < v4.1  | :x:                |

## Reporting a Vulnerability

We take the security of CodeBuddy seriously. If you find a security vulnerability, please do not report it publicly via GitHub issues. Instead, please follow these steps:

1. Send an email to **oyinlolaolasunkanmi@gmail.com** with the subject "Security Vulnerability".
2. Provide a detailed description of the vulnerability, including steps to reproduce it.
3. Include any potential impact and mitigation strategies if known.

We will acknowledge your report within 48 hours and provide a timeline for a fix if applicable. We follow a responsible disclosure policy.
